# Changelog

## NEXUS - AI-Powered Job Matching Platform

All notable changes to this project will be documented in this file.

---

## [1.0.0] - December 2024

### Added

#### Core Features
- User authentication with email/password and Google OAuth
- Role-based access control (Job Seeker, Admin)
- JWT-based session management
- Responsive design for all devices

#### Job Seeker Features
- Personal dashboard with application overview
- Job search with filters (type, location, experience)
- Job application submission
- Resume upload and management
- Application status tracking
- AI-powered cover letter generation
- AI job match analysis
- AI career assistant chatbot

#### Admin Features
- Admin dashboard with analytics
- Job posting management (CRUD)
- Application review system
- Applicant profile viewing
- Resume and cover letter access
- Application status updates
- User management

#### AI Integration
- OpenAI GPT-4o-mini integration
- Automatic cover letter generation
- Resume-job matching with scoring
- Interactive career guidance chatbot

#### Technical Features
- MongoDB Atlas cloud database
- Express.js REST API
- React 18 frontend
- Bootstrap 5 styling
- Vercel frontend deployment
- Render backend deployment
- Google Analytics integration
- Vercel Analytics integration

### Security
- Password hashing with bcrypt
- JWT token authentication
- CORS configuration
- Input validation
- File upload validation
